### BLE (Blueetooth low energy)
very low power consumption
low bandwith
fast setup


ISM band from 2.402 GHz to 2.480 GHz


First Step in establishing a bluetooth connection is pairing , where one device makes itself discoverable and broadcast its precense to other bluetooth device in range
The second device upon receiving this broadcast sends a paring request to the bluetooth that initiated this broadcast, These devices authenticate each other using link key or long-term key. Once paring is done the bluetooth devices need not pair again if they want to communicate in future as the device details are already stored in each device.

Bluetooth devices form a communication ring called piconet , where there is a main device 

Synchronous Connection-Oriented (SCO) links: Primarily used for audio communication, these links reserve slots at regular intervals for data transmission, guaranteeing steady, uninterrupted communication ideal for audio data.
Asynchronous Connection-Less (ACL) links: These links cater to transmitting all other types of data. Unlike SCO links, ACL links do not reserve slots but transmit data whenever bandwidth allows.
